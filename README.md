bootstrap-narrow
================

業務系システムなど、画面項目数が多い画面のため、PaddingとMarginを必要最低限に絞ったBootstrap用テーマです。

bootstrap-narrowテーマCSSをHTMLに追加するだけで使用できます。

Bootstrap対応バージョン
----
bootstrap-narrowでは以下のBootstrapバージョンに対応しています。

2.2.2
